Worker Message Protocol

Message (→ Worker)Response Status (← Worker)Description
{ type: 'check' }checknavigator.gpu.requestAdapter() · shader-f16 feature check
{ type: 'load' }loading → progress → ready / download_retry / download_errorAutoProcessor.from_pretrained + Gemma4ForConditionalGeneration.from_pretrained (q4f16, webgpu)
{ type: 'generate' }start → update* → complete / errorSimple prompt completion (used for initial dev testing)
{ type: 'task' }task_start → task_update* → task_pass1_complete* → task_complete / task_errorMultimodal task with streaming · two-pass for meeting_minutes/voice_to_email
{ type: 'cancel_task' }—InterruptableStoppingCriteria.interrupt()
{ type: 'interrupt' }—InterruptableStoppingCriteria.interrupt() (for generate mode)
{ type: 'reset' }—InterruptableStoppingCriteria.reset()

Model Details

Model IDonnx-community/gemma-4-E2B-it-ONNX
Runtime@huggingface/transformers v4.0.1 (CDN via jsdelivr)
HardwareWebGPU · Q4F16 quantization
Download~1-2 GB · exponential backoff retry (5 attempts, 2s→32s) · partial resume via Transformers.js cache
Feature gateshader-f16 required (otherwise 'unsupported' error)
